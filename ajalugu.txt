GitHub sai alguse 19. oktoobril 2007. GitHubi Beta faas.
Veebileht lasti välja 2008 aprill. Alatest sellest oli teenus kättesaadav.
Microsoft oli hakkanud üheks suurimaks GitHubi kasutajaks alatest 2012.
Microsoft kuulutas huvi hankida GitHubi 4 juunil 2018 7.5 milrjard dollari eest.
Microsoft kinnitas diili 26 oktoober 2018.
2019 spetember GitHub hankis Semmele koodi analüüsi tarkvara.
2020 veebruar GitHub käivitati indias "GitHub India" nime all.
2020 märts GitHub kuulutas, et nad hangivad npm.
GitHub tegi kõik tuuma teenused tasuta kõigile 14 aprill 2020.
2020 juuli alguse GitHub arhiivi programm asutati.
